# How.by

Web application connecting people to get their needs satisfied.


## Getting Started

### Prerequisities

```
Node.js (Express 4)
MongoDB

Backbone.js (Marionette)
Bootstrap
```

### Installing


first start

```
npm install
```

Run server

```
node server/server.js
```
